# Network Lab

A collection of low-level networking projects written in Rust.

## Projects

### dns-server
Manual implementation of a DNS server including packet parsing, filtering, and caching.

### packet-sniffer (planned)
Raw packet capture and analysis tool.

### firewall (planned)
Experimental firewall engine using rule-based packet filtering.

## Goals

- Deep understanding of networking protocols
- Manual protocol parsing (DNS, TCP, UDP)
- OS-level network programming
- High-performance systems design
